# Simple Form Using Flask

A beginner-friendly Flask web application that allows users to submit their name via a web form and receive a personalized greeting. Perfect for learning Flask basics, form handling, and using templates with static files.

---

## Features

- Simple form submission using Flask  
- Dynamic greeting page with Jinja2 templates  
- Organized folder structure: `templates` and `static` folders  
- Stylish CSS design for a modern look  
- Easy to understand and modify for beginners
